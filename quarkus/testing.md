# Testing

Add the extension ``io.quarkiverse.cucumber:quarkus-cucumber`` to the ``pom.xml`` use cucumber

```xml
<dependency>
    <groupId>io.quarkiverse.cucumber</groupId>
    <artifactId>quarkus-cucumber</artifactId>
    <version>1.0.0</version>
</dependency>
```
